== Description ==

An easy way to place a guarantee box on your Wordpress site.

###Quick Start Guide

* [guarantee box quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/guarantee_box/)

###What you get when you use the Guarantee Box

*  Place a guarantee box with a simple shortcode [guarantee_box id="%postId%"]
*  Put a guarantee box on any post or page.
*  guarantee box headline
*  guarantee box seal (image)
*  guarantee box background color
*  guarantee box message


###Add guarantee box to any post or page

Whether you are writing a post about your company or another company, you can include a well formatted guarantee box.


###Guarantee Box 

Add as many guarantee boxes to your site as you like.  The Guarantee Box lets you create guarantee boxes and save them for when ever you are ready to use them.

###Guarantee Seal
The plugin comes with a default 100% Money Back Guarantee seal.  You can also upload your own guarantee seals.

###Quick Start Guide

* [guarantee box quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/guarantee_box/)


== Installation ==


1. Upload `guarantee box` to the `/wp-content/plugin/` directory
1. Activate the guarantee box through the 'Plugins' menu in WordPress
1. Click **Guarantee Box** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The guarantee box does the programming and design.

= Is there a limit to the number of guarantee boxes I can create? =
No, unlimited guarantee boxes can be created.

= Can the guarantee box be put on my homepage? =
Yes, you can put a guarantee box on your homepage by using the Guarantee Box shortcode on you homepage.

= Can I put guarantee boxes on any post or page? =
Yes, use the guarantee box shortcode, [guarantee_box id="%postId%"] on your pages and posts.
